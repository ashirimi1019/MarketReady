# Demo Script

## 30-Second Hook
"Most career tools are GPT wrappers. This is a verification engine. We compute a Market-Ready Index using federal skill standards, live local demand, and proof density from verified work. Then we audit GitHub repos to prove skill claims with code evidence."

## 2-Minute Click Path
1. Open `Student Guide`.
2. Click **Run Market Stress Test**.
3. Point out:
   - MRI formula label and component cards.
   - `live` vs `snapshot` source badge.
   - market volatility chart and top hiring companies.
4. In **GitHub Proof Auditor**, click **Verify with GitHub**.
5. Show:
   - verified skills list,
   - confidence,
   - source badge/freshness.
6. Click **Generate 90-Day Mission**.
7. Show mission lanes/checklists and **Market Pivot** behavior.

## Failure Contingency Line
"Live provider timed out, switching to signed snapshot from `<timestamp>`."

## Pre-Demo Validation
1. `pytest -q`
2. `powershell backend/scripts/hackathon_validate.ps1`
3. Confirm `backend/validation_report.json` shows overall pass.
